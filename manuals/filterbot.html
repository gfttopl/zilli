1. 🤖 About the bot

@hire_filter_bot automates screening of candidates from freelance/job platforms inside Telegram. It collects a structured анкета, gives a test task, and sends everything to a private admin chat with buttons “Accept / Reject”.

Main features:

  • Step-by-step questionnaire: name, phone, email, portfolio links, detailed “about”.
  • Input validation: rejects gibberish names, invalid emails, too-short answers.
  • Random delay (10–40 min) before offering the test task (human-like flow).
  • Collects Google Drive test link and forwards full profile to admin chat.
  • Admin review buttons; prevents double-click; sends result to candidate.
  • Auto cleanup: profiles without test are deleted after 30 days of inactivity.

2. 🧑‍💻 Candidate flow

  1. Candidate starts with /start.
  2. Bot asks and validates: name → phone → email → portfolio links → “about”.
  3. After анкета is complete, bot waits a random time (10–40 min).
  4. Bot asks: “Ready for test? Yes/No”.
  5. If Yes: bot sends test text (from TEST_TEXT) and requests Drive link.
  6. Candidate sends Drive link; bot confirms review.
💡 Tip: Candidate should set Drive access to “anyone with the link”.
⚠️ Important: After a rejection, the candidate cannot reapply (blocked on /start).

3. 🧾 Admin flow

When a test link is received, the bot sends a full profile to the admin chat with buttons:

Action What happens
✅ Accept Candidate gets “We’re interested, we’ll contact you soon”. Buttons disappear.
❌ Reject Candidate gets rejection message and becomes permanently blocked from reapplying.

4. 💾 Storage & cleanup

  • Files are stored at hosting/candidati/<telegram_id>.txt
  • Profile includes: name, phone, email, portfolio, about, tags/summary (optional), test link, status.
  • If there is no test link and no activity for 30 days → profile is deleted automatically.

5. 🧠 AI features (optional)

  • Answers only hiring-related questions in context (e.g., “What if I have no samples?”).
  • Ignores random/trivial questions (math, unrelated).
  • Can extract summary + skill tags from long “about” text without inventing facts.

7. ❓ FAQ

Why does the test question appear later?

Bot uses a random delay (10–40 minutes) to simulate human processing.

Where do profiles live?

In hosting/candidati/<telegram_id>.txt.
